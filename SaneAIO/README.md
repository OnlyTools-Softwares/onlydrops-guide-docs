# SaneAIO

CURRENTLY SUPPORTED: SK, Myntra



### INSTALLATION AND STARTING UP

#### For Windows:

* Download latest version from [releases-or-downloads](../../releases-or-downloads/ "mention").
* Unzip if required.
* **Double click on the .exe** and make sure the **data** and **images folders** are present.

**For MacOS:**

* Download latest version from [releases-or-downloads](../../releases-or-downloads/ "mention").
* Unzip if required.
* **Double click on the executable** and make sure the **data** and **images folders** are present.
* You might have a "did not find directory error" if you're using an older version, please open a **Terminal Window,** write **cd** and then drag an drop the **FOLDER** onto the terminal window and then press ENTER, now, drag and drop the **executable** onto the window as well.



Once started up, it prompts you for your ONLYACO key, which you can find by using the "**$onlyaco check"** command ([helper-bots.md](../../autocheckout/helper-bots.md "mention")), or just simply logging onto [**OnlyDrops Dashboard.**](https://dashboard.onlydrops.in)&#x20;

![Key Prompt](../../assets/Screenshot%202023-02-14%20at%204.42.38%20PM.png)

Once you've typed out or entered your ONLYACO KEY, press ENTER to continue.

!!! warning
    If you're sure about the key's validity, and it is still remarking it as "Invalid", reset your key using the "**$onlyaco reset**" command.

### NOTE: All features going ahead are common for both Windows and MacOS.
