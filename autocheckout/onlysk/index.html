    OnlySK

    Downloading the tool, i.e, the Browser Extension

    To download the latest version of the extension, click on the link below.

    OnlySK v1.0 [Autofill on ATC]

    Warning

    Make sure you've been provided a license key by the team before trying to set this up, if not, please ask for one if verified.

    Once you're done installing it, the first popup, i.e, auth should look like this:

    Once at this page, fetch your key, and paste it/write it into the box with the "Enter your key to activate" placeholder, when pasted, click on activate.

    Success

    NOTE: You don't need to do this every time, once verified and logged in, it should stay that way as long as you don't change your WiFi (your key is bound to your IP), and has not been revoked.

    SIZE SELECTION OPTIONS

    Restock Mode: Not applicable, choose this option only.

    RECHECK DELAY/REFRESH TIMER

    Has various delays options from the fastest to the safest.

    • Sonic Mode: The fastest, script executes as soon as the last reload is complete (+ 0 seconds).
    • 4x: The most recommended option on most drops, does not get as many "too many request" flags (+1 seconds)
    • 2x: (+2 seconds).
    • 1x: (+4 seconds).
    • Restock Mode: The safest, script executes every few seconds, so as to not send too many requests and possibly destroying your captcha pass probability and/or increasing bot activity. (+ 5 seconds).

    To get to the autofill profile menu, click on the OnlyDrops logo on the top and it will redirect you to the page.

    AUTOFILL ADDRESS

    Has multiple fields to be input accordingly.

    • First Name
    • Last Name
    • Email : The one you wish to use to get updates on.
    • Address Field: All the relevant lines of address except city.
    • City: Enter your residence city.
    • Pincode/Zipcode: Closest area post office pincode.
    • Mobile No.: Preferred contact mobile number.
    • State: Dropdown menu to select state.

    Once filled, don't minimise the extension yet, just click again on the OnlyDrops logo and you'll be back at the home page of the ACO menu. Now, press on save to save your details for the autofill.

    Once set, TURN ON the slider and SAVE the settings again. As soon as you do that, click on save and the extension should minimize itself. That should be all.
